

## LOCAL

```
npm run ganache
```

Compile and deploy on your local blockchain
```
truffle migrate
```

Configure your browser wallet with ganache available accounts and ganache local blockchain
```
http://127.0.0.1:8545 
Network ID: 1337
```

In the client folder, start the frontend
```
npm start
```
